## 一家差点倒闭的公司，如何重新定义钢琴？

ROLI这个名字，在音乐科技圈里可不算陌生。

十几年前，他们推出的Seaboard就是个异类——用硅胶做键盘表面，能感知你手指的每一个细微动作。按、揉、搓、滑，每个动作都能产生不同的音色变化。这种"5D触控"技术在当时简直是黑科技，拿了一堆奖，连索尼都来投资。

但创新不等于成功。Seaboard虽然酷炫,但价格昂贵（最早的Seaboard Grand售价高达2999美元），学习曲线陡峭，普通人根本玩不转。2020年,ROLI甚至一度陷入财务困境,差点就要倒闭了。

但他们没放弃。2024年重组后，ROLI找到了新方向——不再只做小众的专业设备，而是要让音乐学习变得简单、有趣、人人都能上手。于是就有了现在的ROLI Piano系统。

## Seaboard：给创作者的"另类"键盘

除了面向学习者的Piano系列，ROLI还有专为音乐创作者设计的Seaboard系列。

Seaboard的键盘表面是硅胶做的，不像传统琴键那样一个个分开，而是连续的波浪形"键波"。这种设计让你能做出各种传统键盘做不到的表现：手指在键上滑动能做出流畅的滑音，揉动手指能产生颤音，按压力度能改变音色明暗。

这种"5D触控"（Strike、Press、Glide、Slide、Lift）给了你前所未有的表现力。想模拟小提琴的揉弦？在键上轻轻晃动手指就行。想做出吉他推弦的效果？在键上左右滑动。想让合成器pad音色慢慢变亮？增加按压力度就好。

不过要说实话，Seaboard不太适合弹传统钢琴曲。它的学习曲线比较陡，更适合电子音乐制作、电影配乐、实验音乐这些需要独特音色和表现力的场景。
